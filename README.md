# wangEdior-formula

wangEditor公式编辑器kityformula + myscript-math-web


```
# 克隆项目
git clone https://github.com/mirror29/wangEdior-formula.git

# 进入项目目录
cd wangEdior-formula

# 安装依赖
npm install

# 运行项目（将自动在浏览器打开 http://localhost:8080 ）
npm run serve
```

其中，wangEditor 相关的代码可参考 `src/views/Home.vue`
